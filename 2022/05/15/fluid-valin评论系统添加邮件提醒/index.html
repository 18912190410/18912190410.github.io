fluid-valin评论系统添加邮件提醒

添加邮件提醒

首先需要确保你的valin评论功能能用，这个网上有好多教程了

自定义环境变量

在云引擎-WEB-设置里自定义环境变量

在云引擎-WEB-部署，点Git部署
填写代码库：https://github.com/zhaojun1998/Valine-Admin

保存

在下方分支或版本号输入master

注意：

这里推荐代码库路径先填写
https://github.com/zhaojun1998/Valine-Admin
然后全部部署完之后试试留言会不会提醒道邮箱里，如果配置成功，在github上frok这个代码库到自己账号，然后代码库路径使用自己账号下的库，然后改成自己的
https://github.com/用户名/Valine-Admin（为了稳定运行）

部署

设置后台管理登录信息，点击 存储 -> 结构化数据，选择User如果有内容，全部删除

新建后台管理的账户

选择User -> 添加行

填写password、username、email

username 任意即可

email为登录的邮箱，建议为配置中的 SMTP_USER 或 TO_EMAIL

password 作为账号密码

之后就可以通过绑定的域名访问评论管理系统

自动唤醒

（如果你不想花钱的话）

fork 这个仓库到自己的账户里：https://github.com/AimTao/leancloud-self-wake

进入这个仓库，setting-secrets-actions添加一个秘密Name 一定要是 SITE，因为 脚本里要用这个名字；Value 则为你的评论后台网站的地址（https://刚才绑定的域名）

点击你的仓库上方的 Actions 选项，，点击 I understand... 的按钮，确认在 Fork 的仓库上启用 GitHub Actions （可能需要点击一下enable什么的）

点击自己仓库右上角 star ，star 你的仓库，这是为了触发 Github Action 第一次执行，之后就会自动执行定时任务。
